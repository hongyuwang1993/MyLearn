# DNS解析的过程

```sequence
本机->本机host文件: 查询域名ip解析关系
本机host文件->本机: 我没有找到
本机->本机dns缓存: 查询域名ip解析关系
本机dns缓存->本机: 我没有找到
本机->本地dns服务器: 查询域名ip解析关系
本地dns服务器->跟域名服务器: 请求
跟域名服务器->本地dns服务器: 对应的顶级域名服务器
本地dns服务器->顶级域名服务器: 请求
顶级域名服务器->本地dns服务器: 域名的供应商服务器
本地dns服务器->域名供应商服务器:请求
域名供应商服务器->本地dns服务器: 域名对应的ip
Note right of 本地dns服务器: 存储域名ip对应关系
本地dns服务器->本机: 域名对应的ip
```
